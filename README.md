## view-router  和  v-if else   v-show的实践
### view-router  适用于父组件中右固定内容  路由子组件  那么此时 设置index.js就好了
### v-if else v-show  如果同等级的组件需要有切换的需求   这时候view-roter  无法切换  
###  这时候使用v-if  else  同等级的组件完成对应的处理之后 要emmit 事件给父组件
